PSK-AM

PSK-AM is a very narrow-band and reliable amateur mode developed by F6CTE. The aim of this experimental slow mode is to allow chat communication between hams.

Parameter

Value

Frequency range

HF

Operation modes

Simplex

Modulation

DBPSK

Symbol rate

10, 31.25 and 50 Bd

Receiver settings

DATA, CW, LSB or USB

Input format(s)

AF, IF

Additional Info

14.075 MHz

PSK-AM is similar to PSK-31, but uses its own fixed length 8 bit alphabet with a 5:3 ratio. Each character is repeated 5 positions later. This diversity in time allows the correction of an invalid character.

Bitstream:

D(x)

R(x-2)

D(x+1)

R(x-1)

D(x+2)

R(x)

D(x+3)

D(x):    character

R(x):     repeated character
