BR-6028

BR-6028 is a VFT (Voice Frequency Telegraph) frequency and time diversity modem using 7 data channels. It is sometimes also known as “BARRIE” or “USA-7” channel modem.

Parameter

Value

Frequency range

HF

Operation modes

Baudot, ASCII

Modulation

VFT FSK

Symbol rate

45.45, 50, 75, 100, 103.7 Bd, variable 30-110 Bd

Pilot Tone

560 Hz

Receiver settings

DATA, CW, LSB or USB

Input format(s)

AF, IF

Theoretically the system may handle channel dropouts of up to 3 seconds before errors occur. An un-modulated pilot tone is transmitted at 560 Hz.
