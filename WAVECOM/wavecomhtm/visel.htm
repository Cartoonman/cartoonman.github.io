VISEL

VISEL, also known as FEC-12 or YUG-MIL, is a synchronous system. This mode is used by defense forces in Eastern Europe.

Parameter

Value

Frequency range

HF

Operation modes

Simplex

Modulation

FSK

Shift/Bandwidth

300 Hz

Symbol rate

81.3, 120.9, 123.5, 125 Bd

Receiver settings

DATA, CW, LSB or USB

Input format(s)

AF, IF

Additional Info

ITA-2

The unusual baud rate of 120.9 Bd is very typical for this mode. Synchronization is done at the beginning of every burst transmission. The unencrypted calling sequence with tactical call signs has an autocorrelation of 12 bits and all bits are interleaved. All traffic is encrypted. Occasionally operator chat in plain text is transmitted.
